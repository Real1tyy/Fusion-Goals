# Header Actions

The Fusion Goals view header displays quick-action buttons for common operations.

## Default Actions

- **Create Goal** — Opens the goal creation modal
- **Create Task** — Opens the task creation modal

## Customizing Actions

Click the gear icon in the header to open the **Manage Header Actions** modal. From there you can:

- **Reorder** actions by dragging or using the arrow buttons
- **Rename** an action to a custom label
- **Change icon** to any Lucide icon name
- **Change color** to a custom accent color
- **Hide** actions you don't use
- **Reset** any customization back to the default

All customizations persist across sessions.

## Command Palette

You can also open the action manager via the command palette:

- **Fusion Goals: Manage header actions**
