# CRUD Modals

Create and edit goals and tasks directly from modal dialogs without leaving your current view.

## Create Goal

Use the **Create Goal** command to open a modal with fields for:

- **Name** — The goal file name
- **Status** — Dropdown with your configured status values
- **Priority** — Critical, High, Medium, or Low
- **Start Date** — Date picker
- **End Date** — Date picker
- **Progress** — Percentage (0-100)

## Create Task

Use the **Create Task** command to open a modal with the same fields as goals, plus:

- **Goal** — Dropdown showing all existing goals to link the task to

## Create Task from Goal

When viewing a goal file, use the **Create Task from Goal** command to create a task pre-linked to that goal with inherited properties.

## Editing

Open the same modal with an existing goal or task to edit its properties. Changes are saved directly to the frontmatter.
