# Priority & Progress

Goals and tasks support two additional frontmatter properties for tracking importance and completion.

## Priority

The Priority property accepts four levels:

- **Critical**
- **High**
- **Medium**
- **Low**

The property name is configurable in settings (default: `Priority`). Priority values appear in the dashboard charts and Bases table columns.

## Progress

The Progress property tracks completion as a percentage from 0 to 100. Set it manually in frontmatter or via the CRUD modal.

The property name is configurable in settings (default: `Progress`). Progress distribution is shown in the dashboard chart.

## Configuration

Go to **Settings → Hierarchy → Additional Properties** to change the property names used for Priority and Progress.
