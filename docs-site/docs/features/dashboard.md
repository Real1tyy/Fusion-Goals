# Dashboard & Tabs

Fusion Goals uses a multi-tab interface to give you a complete project management overview.

## Tabs

The main view contains four tab groups:

- **Dashboard** — Interactive grid of pie charts showing data distributions
- **Goals** — Obsidian Bases tables for goals, sorted by name, status, priority, or deadline
- **Tasks** — Obsidian Bases tables for tasks, sorted by name, goal, status, priority, or deadline
- **Graph** — The interactive relationship graph showing goal-task hierarchies

Use the tab bar at the top to switch between views, or use keyboard commands for quick navigation.

### Editable Tabs

Right-click any tab to rename, reorder, or hide it. Click the gear icon on the tab bar to open the tab manager where you can manage all tabs at once. Your tab customizations — including the active tab, renames, visibility, and order — persist across sessions.

## Dashboard

The Dashboard tab displays a 2x3 grid of pie charts:

- **Goal Status** — Distribution of status values across goals
- **Task Status** — Distribution of status values across tasks
- **Goal Priority** — Priority distribution for goals
- **Task Priority** — Priority distribution for tasks
- **Deadlines** — Deadline proximity (Overdue, This Week, This Month, Later, No Deadline)
- **Progress** — Progress distribution across all goals and tasks

### Swapping Charts

Each grid cell has a swap button. Click it to open a picker showing all available chart types. Select a different chart to replace the current one.

### Enlarging Charts

Click the enlarge button on any chart cell to view it in a full-size modal.

### Persistent Layout

Your grid layout is saved automatically. Swap cells and they stay in place across sessions.
